# Implementação de Matrizes Esparsas

Este projeto implementa e compara duas estruturas de dados para matrizes esparsas: **Tabela Hash** e **Árvore AVL**. O sistema inclui testes automatizados para validar as operações de inserção, multiplicação, soma e transposta.

## Pré-requisitos

* Compilador **GCC**
* Ferramenta **Make**

## Instruções de Compilação

O projeto utiliza um `Makefile` para gerir a compilação e a criação do diretório de binários.

Para compilar ambas as implementações (Hash e AVL) e gerar os executáveis na pasta `bin/`, execute o comando:

```bash
make all
